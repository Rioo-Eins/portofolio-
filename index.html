TENTANG SAYA

MUHAMMAD RIO

Saya adalah seorang remaja yang duduk di bangku kelas 10 tingkat sekolah menengah kejuruan, saya bersekolah di SMK Negeri 1 Batam dengan mengambil jurusan TKJ(Teknik Komputer dan Jaringan). Hobi saya adalah bermain Game mobile, menggambar, mengeksplorasi hal baru, menambah value diri dengan membaca buku filsuf atau pengetahuan teori. Tujuan saya membuat portfolio ini adalah untuk mengenalkan diri saya. Ya, saya memang jurusan TKJ di mana seharusnya fokus di instalasi jaringan, tapi bukan berarti saya tidak bisa menguasai hal lain saya menguasai beberapa hal bidang IT contohnya Desain, Rekayasa perangkat lunak, menggunakan Office, dan saya masih akan menambahkan skil saya di hal lain seperti Cyber Security dan Desain 3D. Ini adalah kisah awal saya dalam membangun masa depan, jadi ketika saya dapat mengusai hal baru saya akan menambahkannya di sini sebagai pengingat.

KEAHLIAN

Website Design

Saya bisa mendesain website sederhana dengan menggunakan bahasa pemrograman HTML, CSS dan sedikit Javascript. pernah menggunakan Boostrap untuk membuat navbar, saya menyukai hal tentang keindahan tampilan. Imajinasi saya cukup besar jadi saya selalu mengarahkannya ke hal seni, di mulai dari menggambar, mewarnai, melukis, mendisain PowerPoint dan kini mendisain Website Biasanya saya mencari inspirasi dengan menonton sebuah film atau menggabungkan beberapa BOM Idea yang terlintas

Cyber Security

Cyber Security adalah bidang yang ingin saya asah selanjutnya, memang saat ini saya belum terlalu mengenal cyber tetapi saya memiliki impian untuk menambahkan bidang satu ini ke dalam skil saya. Bergerak di bagian keamanan internet memiliki gaji yang cukup besar, makanya saya ingin menambahkan bidang ini dalam skil saya, memang membutuhkan waktu yang lama untuk benar benar menguasainya tapi bukan berarti mustahil. Ketika saya berhasil menguasai Cyber Security saya akan mengubah tulisan ini menjadi "Yes i do", jadi bersabarlah menunggu saat itu

Microsoft Office

Saya juga menguasai beberapa Microsoft Tools seperti Word, PowerPoint dan Excel, meskipun di Excel saya masih baru belajar tapi kalau di PowerPoint dan Word saya yakin "Yes I Do". Seperti yang saya katakan di awal saya memiliki imajinasi yang besar jadi ketika saya membuat sebuah PowerPoint saya menjadikannya menarik dan interaktif

Desain 3D

3D Design ini adalah bidang yang saya ingin kuasai dari dulu, karena saya sering menggambar 2D saya ingin mengaplikasikannya ke 3D design untuk memuaskan rasa ingin tahu saya. Tidak pernah memulai belajar dari dulu karena laptop saya yang tidak mendukung 3D graphic karena tidak memiliki VGA, jadi hanya bisa belajar teori melalui YouTube tanpa praktek. Tapi sekarang saya memiliki laptop yang memiliki VGA walau terbilang belum mencukupi tetapi bisa untuk memulai belajar 3D design karena kapan lagi mau belajar kalau bukan sekarang, benar bukan?
